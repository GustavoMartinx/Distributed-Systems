PARA O JAVA:
Baixar o plugin do grpc em : https://central.sonatype.com/artifact/io.grpc/protoc-gen-grpc-java/1.62.2/versions 
CLICA EM BROWSE que te leva para https://repo1.maven.org/maven2/io/grpc/protoc-gen-grpc-java/1.62.2/
ai seleciona a versão para o sistema

coloca o plugin no mesmo lugar do protoc, ou faz os esquemas de configurar PATH, aqui coloquei nam esma pasta
ai usa o comando : protoc --java_out=<DESTINO> --grpc-java_out=<DESTINO> --plugin=protoc-gen-grpc-java=<CAMINHO DO PLUGIN GRPC> MoviesRPC.proto

pro python é só instalar as dependencias e fazer igual ta nos slides

para rodar o java usa mvn clean compile exec:java (executa e compila)
ou mvn exec:java -D"exec.mainClass"="MovieClient" (só executa)
